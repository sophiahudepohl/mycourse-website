Two-Column Layout

The sidebar is floated left; the main content is floated right. The parent .wrap uses overflow:auto; to contain floats.

Clearing Strategy

Because the parent contains its floats, the footer won’t overlap the columns. This is a classic clearfix approach covered in Chapter 6.
